Project 2 Street Fighter

Created by Abdul Qureshi

This is a two-player game made in P5.js, Arduino, and P5.play to animate sprites and control them using p5.serialControl to enable a hand-held circuit that would control one player. In order to win, a player must decrease the health of the other to 0 or vice versa using a character's abilities.
